Model Architecture Upgrades

  • Byte-Pair Encoding (BPE) Tokenizer Vocab: 1024
    Replaced the raw byte-level tokenizer with an optimized BPE tokenizer trained on the entire corpus. Achieve 2.21x sequence compression, which effectively doubles the model's context window (block size 128 tokens represents ~283 bytes of text). Implemented cache lookup and quick membership checks to process the 7.3 MB training set in 15 seconds.
  • Rotary Positional Embeddings (RoPE) Parameter-free
    Replaced absolute learned positional embeddings with Rotary Position Embeddings. Improved relative position modeling and saved parameters to reallocate to model capacity.
  • Weight Tying Saved 164k params
    Tied weights of the token embedding matrix and the language model output head, reducing parameters from 1.55M to 1.39M, enabling a larger vocabulary size.
  • SwiGLU Feed-Forward Blocks d_ff: 426
    Substituted GELU activations with LLaMA-style SwiGLU feed-forward networks (without biases), offering smoother gradients and quicker learning rates.
  • RMSNorm Normalization Simplified Norm
    Replaced LayerNorm with parameter-free RMSNorm (with learned scaling weights only), speeding up CPU computation per step.
  • AdamW & Cosine LR Schedule Warmup: 200 steps
    Implemented a cosine decay schedule (peak 1e-3, min 1e-4) with a linear warmup phase. Utilized AdamW optimizer with a weight decay of 0.1 (excluding biases and norms) and gradient clipping at 1.0.
